@startuml
participant "Dataset Preparation Script" as DPS
participant "Dataset Splitter" as SP
participant "Augmentation" as AC
database "File System" as FS

DPS -> SP: Split balanced dataset into train,\nvalidation, and test sets
activate SP

SP -> SP: Perform first stratified split\ninto temporary set and test set\naccording to test ratio
SP -> SP: Compute adjusted validation ratio\nfor remaining data
SP -> SP: Perform second stratified split\nof temporary set into train and validation

SP --> DPS: Provide train, validation, and test\nimages and labels
deactivate SP

DPS -> AC: Balance training set via augmentation\n(minority classes only)
activate AC

AC -> AC: Count samples per class in training labels
AC -> AC: Determine augmentation target\n(maximum class size among training classes)

loop For each class label
    AC -> AC: Get number of training samples\nfor this class
    alt Class has fewer samples than target
        AC -> AC: Compute how many augmented samples\nare needed
        AC -> AC: Select original training images\nof this class as sources

        loop For each required augmented sample
            AC -> AC: Choose a random original image\nof this class
            AC -> AC: Add small Gaussian noise\nand clip to valid pixel range
            AC -> AC: Store augmented image and label
        end
    else Class already at or above target
        AC -> AC: No augmentation for this class
    end
end

AC -> AC: Concatenate original and augmented\ntraining images and labels
AC -> AC: Shuffle final training set\nwith a random permutation

AC --> DPS: Return final training images and labels
deactivate AC

DPS -> FS: Save final train, validation,\nand test sets
FS --> DPS: Confirm all dataset files written

@enduml
