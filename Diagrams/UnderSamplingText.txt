@startuml
participant "Dataset Preparation Script" as DPS
participant "Data Loading and Filtering" as DF
participant "Balancing" as BC
database "File System" as FS

DPS -> DF: Begin loading and filtering raw dataset
activate DF

DF -> FS: Load raw image and label arrays\nfrom input .npy files
FS --> DF: Raw images and labels in memory

DF -> DF: Compute per-sample density\n(fraction of non-zero pixels)
DF -> DF: Build mask for samples with density\n>= minimum density threshold
DF -> DF: Apply mask to obtain filtered images\nand filtered labels

DF --> DPS: Return filtered images, labels,\nand corresponding densities
deactivate DF

DPS -> BC:  Undersample dominant classes
activate BC

BC -> BC: Identify all unique class labels

loop For each class label
    BC -> BC: Collect indices and densities\nfor samples of this class
    BC -> BC: Count available samples

    alt Class has more samples than target per class
        BC -> BC: Sort class samples by density (descending)
        BC -> BC: Select top samples up to target balance\n(keep densest examples)
    else Class has target or fewer samples
        BC -> BC: Keep all samples for this class
    end

    BC -> BC: Add selected indices to global list\nof balanced indices
end

BC -> BC: Use balanced indices to build\nbalanced images and balanced labels
BC --> DPS: Return images and labels
deactivate BC

@enduml
